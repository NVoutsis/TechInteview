Στο αρχείο excel με τίτλο "x_y_coords_to_check", δίνεται λίστα με συντεταγμένες σε μη δεδομένο σύστημα αναφοράς και μη ελεγμένες ως προς την εγκυρότητά τους.
Επιπλέον δίνεται γεωβάση (PREFECTURES_4326.gdb) με τις περιφερειακές ενότητες σε σύστημα αναφοράς WGS84.

Να αναπτυχθεί ένα γεωχωρικό εργαλείο (geoprocessing tool) σε python με τη χρήση της βιβλιοθήκης arcpy

1. Να ελεγχθεί η ομοιογένεια των δεδομένων και να διορθωθούν όπου αυτό είναι δυνατόν. (Γράμματα αντί για ψηφία, πιθανώς άκυροι αριθμοί ή αντεστραμμένες συντεταγμένες)
2. Να ελεγχθεί αν οι συντεταγμένες είναι ΕΓΣΑ87 ή WGS84, ή τίποτα από τα δύο.
3. Να δημιουργηθούν δύο Feature Classes, ένα με τα σημεία σε ΕΓΣΑ87 και ένα με τα σημεία σε WGS84.
4. Να δημιουργηθεί ένα πεδίο σε κάθε Feature Class με την Περιφερειακή Ενότητα στην οποία εμπίπτει το κάθε σημείο, σύμφωνα με το δεδομένο Feature Class Περιφερειακών Ενοτήτων.
5.(Προαιρετικά) Να γίνει μια Density Based Clustering ανάλυση και να δημιουργηθεί ένα νέo feature class με τα αποτελέσματα.

Τα αποτελέσματα να αποθηκευτούν στη γεωβάση "workspace".
